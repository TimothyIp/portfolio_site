My Work

Project 01.

ShowSherpa

React - API - Firebase - Javascript/ES6 - Gulp - HTML5 - CSS3 - Sass

ShowSherpa is a React application that enables users to sign in through their Google accounts and lets them save their currently watched TV Shows into the their collection. This information is then shown on a calendar with upcoming episode time and dates for reference.

Project 02.

BaseGrounds

Javascript/ES6 - jQuery - API - Gulp - HTML5 - CSS3 - Sass

BaseGrounds lets users search different city locations to gather information about the startup scene and momentum in that city. This is done by searching through scities and making calls to the Teleport API for each city's data.

Project 03.

EncryptYou

Javascript - jQuery - Gulp - HTML5 - CSS3 - Sass

EncryptYou is a message encrypter website, where users type in a message and then they choose a picture to associate a key to that message which then scrambles it. The only way to decrypt that message is to select the same picture which was choosen at the time of encryption.

Project 04.

PSD To HTML Conversion

HTML5 - CSS3 - Sass - Gulp

PSD template conversion into a full working website. Fully responsive with complete layout changes when site is resized.

Project 05.

Client Website

HTML5 - CSS3 - Gulp - Sass - Javascript - jQuery

Was giving a design mockup from the client and they requested that no frameworks or external libraries were to be used. So the image gallery and carousel are coded from the ground up.
