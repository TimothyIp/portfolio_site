My Work

Project 01.

Real Time Chat App

React - Socket.io - Express - MongoDB - Node.js - Javascript/ES6 - HTML5 - CSS3

Full-stack application using React and Node.js that lets users chat with other users in real-time using Socket.io. Users can create persistent channels and private message other users. Message logs are saved using MongoDB.

Project 02.

ShowSherpa

React - API - Firebase - Javascript/ES6 - Gulp - HTML5 - CSS3 - Sass

ShowSherpa is a React application that enables users to sign in with their Google accounts and lets them save their currently watched TV Shows into their collection of shows. This information is then shown on a calendar with upcoming episode time and dates for reference.

Project 03.

BaseGrounds

Javascript/ES6 - jQuery - API - Gulp - HTML5 - CSS3 - Sass

BaseGrounds gives users the ability to search for different city locations to gather information about the startup scene and momentum in that specific city. This is done by making ajax calls to the Teleport API for data on that city and displaying the relevant startup information.

Project 04.

EncryptYou

Javascript - jQuery - Gulp - HTML5 - CSS3 - Sass

EncryptYou is a message encrypter project, where users type in a message and then they get to select a picture to associate a key to that message which then scrambles it. The only way to decrypt that message is to select the same picture which was choosen at the time of encryption.

Project 05.

PSD To HTML Conversion

HTML5 - CSS3 - Sass - Gulp

PSD template conversion into a fully working website using responsive design methodology.
